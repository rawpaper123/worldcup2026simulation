# WorldCup2026 Simulation

Open-source microsite for the World Cup 2026 bounded world model simulation.

This repository is intentionally separate from the private `rawpaper123/nira` repository. It contains only the standalone public campaign page, scoped styles, and typed mock data needed to run the visual simulation locally.

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Scope

- Visual campaign page based on the final HTML prototype.
- Typed mock schedule, simulation, agent, and log data.
- No real match API, prediction backend, or social export integration.
